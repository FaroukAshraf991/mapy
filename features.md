# Mapy Future Roadmap

This document tracks all currently planned and possible future features for the Mapy application. These are ideas to be implemented upon explicit user request.

## 🛣️ Navigation & Routing
1.  **Multi-Stop Routing**: Support for intermediate waypoints (A -> B -> C) during a single trip.
2.  **Voice Guidance**: Integration of Text-to-Speech (TTS) for turn-by-turn spoken instructions.
3.  **Search Along Route**: One-tap searching for gas stations, restaurants, or parking while navigation is active.
4.  **Toll & Traffic Avoidance**: Profile-based routing to bypass tolls or heavy congestion.
5.  **Multi-Route Preview**: Showing 2-3 alternative routes with time/distance differences before starting.

## 🗺️ Map Experience
6.  **Offline Maps (Vector Cache)**: Allowing users to download specific regions for navigation without internet connectivity.
7.  **3D Terrain & Building Models**: Visualizing building extrusions and topography for a more immersive 3D experience.
8.  **Traffic Heatmap Layer**: Real-time traffic congestion visualization via open data feeds.
9.  **Social/Shared Pins**: Ability to share a custom pin or live location with another registered Mapy user.

## 👤 User & Account
10. **Saved Collections**: Organize Home/Work and other favorite pins into named folders (e.g., "Favorite Parks," "Client Sites").
11. **Trip History & Analytics**: View a timeline of past trips with distance and time statistics.
12. **Real-Time ETA Sharing**: Generate a temporary web-link or internal message to share your live ETA and location with others.

## 🛠️ Specialized Features
13. **Public Transport Profiles**: Integrating GTFS or public transit routing for bus, rail, and subway navigation.
14. **Custom Map Styles**: Allow users to import or select custom Mapbox/MapLibre style JSONs.
15. **Dashcam Integration**: Support for using the phone camera as a Dashcam during active navigation.
