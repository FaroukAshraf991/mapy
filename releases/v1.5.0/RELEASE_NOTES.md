# Release Notes: Mapy "Aetheric Glass" v1.1.0

This release introduces the **"Aetheric Glass"** premium UI revamp, providing a high-fidelity, modern experience with fluid motion and full Flutter 3.41.5+ compatibility.

## ✨ Key Features & Improvements

### 🎨 "Aetheric Glass" Design System
- **Glassmorphic Overlays**: Applied high-end `BackdropFilter` with `ImageFilter.blur` across the Search Bar, Location Chips, and Navigation Guidance.
- **Neon-Ghost Aesthetic**: Refined borders and shadows for a premium, translucent look in both light and dark themes.
- **Unified Icons**: Standardized navigation and map controls for visual consistency.

### 🚗 Premium Navigation Experience
- **Fluid Motion**: Integrated **Hero animations** for smooth transitions between search and active guidance.
- **Refined Camera**: Optimized 3D camera dynamics and location tracking for ultra-smooth map interactions.
- **New 2D/3D Indicators**: Intuitive icons for switching perspectives (`Apartment` for 3D, `Map` for 2D).

### 🛠️ Performance & Reliability
- **SDK 3.41.5+ Migration**: Fully migrated to the `withValues` API, resolving critical "Main Isolate Paused" regressions.
- **Zero-Error Codebase**: Achieved a clean state with 0 lint warnings and optimized native map layer updates.
- **Removed Terrain Style**: Streamlined the UI by focusing on Street and Satellite views.

## 📦 Technical Details
- **Build**: Release APK
- **Flutter SDK**: 3.41.5+
- **Dart SDK**: 3.6.0+
- **Stability**: Verified on multiple device themes.
